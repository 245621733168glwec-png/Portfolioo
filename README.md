# Personal Portfolio Website

This is a single-page portfolio website designed to showcase my skills, services, and background as a web developer. It features a dark-themed design with custom visualizations for skill proficiency.

## Quick Start
No installation is required since this is a static site.

1. Download or clone this folder.
2. Ensure `index.html` and `style.css` are in the same directory.
3. Open `index.html` in your web browser.

## Tech Stack
*   **HTML5** - Structure and semantic layout.
*   **CSS3** - Custom styling, including radial gradients for skill bars and a cohesive navy color scheme.
*   **FontAwesome** - Used for social media and interface icons.

## Key Features
*   **Skill Visualization:** Displays technical proficiency in HTML, CSS, JavaScript, and Python using both linear and radial progress bars.
*   **About Section:** A detailed table layout for personal information and biography.
*   **Services Grid:** distinct sections highlighting Web Development, Creative Design, and other services.
*   **Contact Layout:** Includes a styled form and contact details (Phone, Email, Address).

## Project Structure
*   `index.html`: The main markup file.
*   `style.css`: Contains all visual styling, resets, and layout logic.
*   `script.js`: Linked for future interactivity.
